==========
知乎 Stat
==========

Retrieves a Zhihu user's profile for basic stats

## Installation and Deployment
* Clone the repo
* Run `npm install`
* Run `nodemon app.js` 
* Try the app at `localhost:3000`

## Structure

NodeJS + Express ⇄ AngularJS + Bootstrap
