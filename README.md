zhihu-stat
==========

Scrapes a Zhihu user profile for basic stats

### Example
`node zhihu-user.js brianz`

```Javascript
{
  "name": "Brian",
  "location": "Washington, DC",
  "following": "43",
  "followers": "13308",
  "questions": "8",
  "answers": "104",
  "essays": "4",
  "bookmarks": "1",
  "edits": "457",
  "profile_views": "6440"
}
```
